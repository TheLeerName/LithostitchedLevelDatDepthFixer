# Lithostitched level.dat Depth Fixer
Server-side Minecraft mod which fixes bug in Lithostitched + TerraBlender when loading world creates a loop of data leading to game crash (nbt depth exceeds 512), this is done by mixin in LevelStorageAccess.getDataTag which removes `level.dat` tag in path `Data.WorldGenSettings.dimensions.minecraft:overworld.generator.settings.surface_rule.sequence[1]` before each world load
